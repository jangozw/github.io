docker 搭建开发环境

Posted on 2018-10-05 | Edited on 2018-10-11

命令

命令解释

在后台开启一个docker容器，镜像是已经保存过的 jangozw/centos-web.
-p 设置端口映射，将宿主机的端口 映射到容器的端口，相当于在宿主机访问这个端口就到了容器里面。如上我将12000端口映射到80，意思是宿主机的12000端口访问直接到里容器的nginx服务器，也可以两边端口一样的。

-v 是自动同步，将宿主机的某个目录同步拷贝到容器里的路径，中间用冒号隔开。比如说项目文件肯定在宿主机的，这时候做个同步很实用。

安装环境

在容器里安装好php，nginx 环境，配置nginx server，hosts，还有上面命令里的目录 按照独立主机方式配置就行，没什么特别注意的，最后要将容器的修改提交保存，这不能忘。如果run后发现参数不对，就docker rm 容器名 ,删除容器，在重新docker run 就好了，记得一定要docker commit

测试

执行上面命令，获得容器id，进入容器，开启nginx，php-fpm 进程。

浏览器访问：

localhost:12000

发现已经访问到了容器里的文件，这就对了，不对就重新按照上面步骤检查

提醒

在容器里的一切操作，不想丢的话，一定要commit，push保存，很容易忘记。

后记

docker 容器设置开机启动服务，折腾失败，然后写脚本手动执行的。每次启动容器 /etc/hosts 文件都会被清掉，但这个不影响，可以直接宿主机hosts

阿里云ssh登陆不上问题

Posted on 2018-09-21

问题描述

⇒  ssh root@59.110.219.135
ssh_exchange_identification: read: Connection reset by peer

解决问题

  • 网上找答案

    全是没卵用的

  • 在阿里云提工单

    完美解决，原因：这个可能是您的本地地址访问服务器较多，这边云盾认为有可能是攻击行为，所以将这个地址拦截了一段时间，这个添加白名单之后正常就不会有拦截了

    解决方法：设置阿里云访问的白名单，白名单ip是自己本地的公网ip（可百度 本机ip 查询到）

    设置白名单链接：https://yundun.console.aliyun.com/?p=sc#/sc/visitWhiteList
