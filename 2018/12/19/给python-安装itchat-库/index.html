给python 安装itchat 库

Posted on 2018-12-19

这个应该结合上一个安装python的帖子一起看，最好安装时候给解决掉

安装命令:

[root@1ecde053bfc2 python-3.7]# pip install itchat

遇到问题：

原因：
需要先安装ssl

解决步骤：

1，查看openssl 是否安装

[root@localhost ~]# rpm -aq|grep openssl

yum安装openssl-devel

[root@localhost ~]# yum install openssl-devel -y

执行注意看结果，如果有报错说某个py文件语法错误，很可能是解释头不对，
因为我们安装时候用python 高版本 替换了自带的python2.7。而那个文件需要用2.7的解释头。所以就去那个文件修改下头文件即可，
比如：vi /usr/libexec/urlgrabber-ext-down

然后重新执行yum install openssl-devel -y 即可

2， 重新编译安装 python

./configure –prefix=/usr/local/sbin/python-3.7 –with-ssl

make && make install

安装完成后重新执行

[root@1ecde053bfc2 python-3.7]# pip install itchat

然后发现问题已经解决：
