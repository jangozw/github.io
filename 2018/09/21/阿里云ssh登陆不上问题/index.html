阿里云ssh登陆不上问题

Posted on 2018-09-21

问题描述

⇒  ssh root@59.110.219.135
ssh_exchange_identification: read: Connection reset by peer

解决问题

  • 网上找答案

    全是没卵用的

  • 在阿里云提工单

    完美解决，原因：这个可能是您的本地地址访问服务器较多，这边云盾认为有可能是攻击行为，所以将这个地址拦截了一段时间，这个添加白名单之后正常就不会有拦截了

    解决方法：设置阿里云访问的白名单，白名单ip是自己本地的公网ip（可百度 本机ip 查询到）

    设置白名单链接：https://yundun.console.aliyun.com/?p=sc#/sc/visitWhiteList
