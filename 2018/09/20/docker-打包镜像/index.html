docker 打包镜像

Posted on 2018-09-20

Docker 打包命令

拉取镜像

⇒  docker pull centos

创建容器

⇒  sudo docker run -it --name myFirstCentos centos /bin/bash

查看容器

⇒  docker ps -a
CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS                      PORTS               NAMES
4da46cd0a26a        centos              "/bin/bash"              9 days ago          Exited (0) 40 minutes ago                       myFirstCentos

启动容器

⇒  docker start myFirstCentos
myFirstCentos

进入容器 （添加test.txt）

⇒  docker attach myFirstCentos
    [root@4da46cd0a26a /]# ls
    bin  dev  etc  home  lib  lib64  media  mnt  opt  proc  root  run  sbin  srv  sys  tmp  usr  var
    [root@4da46cd0a26a /]# cd tmp/
    [root@4da46cd0a26a tmp]# ls
    ks-script-Lu6hIQ  test.txt  yum.log
    [root@4da46cd0a26a tmp]# exit

提交修改

⇒  docker commit 4da46cd0a26a jangozw/centos-test:v1
sha256:783ce3eaff7129a839cc67cf6bd88b6effc9ac6f9b2a36e5508c07d6666e1cd1
⇒

登陆docker 

⇒ docker login (执行会让你输入用户名密码)

推送到hub.docker 

⇒  docker push jangozw/centos-test:v1
The push refers to repository [docker.io/jangozw/centos-test]
0b2b943ab4e1: Pushed
1d31b5806ba4: Mounted from jangozw/centos-lnmp
v1: digest:         sha256:81651353f780acd8842be300ca9c0cdeea93f87ae016b76925cdbfd00562c984 size: 736
⇒

提交成功，查看镜像

⇒  docker images
REPOSITORY            TAG                 IMAGE ID            CREATED             SIZE
jangozw/centos-test   v1                  783ce3eaff71        3 minutes ago       200MB
jangozw/centos-lnmp   v1                  09d5dd7b4fd3        About an hour ago   200MB
nginx                 latest              06144b287844        2 weeks ago         109MB
centos                latest              5182e96772bf        6 weeks ago         200MB
⇒

在远程仓库查看

https://hub.docker.com

打包成功，这时候可以拉取自己的镜像了

⇒ docker pull jangozw/centos-test:v1

创建这个镜像的容器后发现 test.txt 在里面
